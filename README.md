# Simple Interest Calculator

A simple command-line tool to calculate simple interest.

## Formula

```
Simple Interest = (P × R × T) / 100
```

Where:
- **P** = Principal amount (initial investment)
- **R** = Annual rate of interest (in percentage)
- **T** = Time period (in years)

## Usage

```bash
python simple_interest.py
```

Enter the values when prompted:
- Principal amount
- Rate of interest
- Time period (years)

## Example

```
Enter principal amount: 1000
Enter rate of interest: 5
Enter time period (years): 2

Simple Interest = 100.0
Total Amount = 1100.0
```

## Features

- Calculates simple interest accurately
- Handles decimal inputs
- Displays both interest and total amount
- Input validation for non-negative values
